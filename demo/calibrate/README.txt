This directory contains scripts like those I use to calibrate my display.

cal_server.py -- Run this first.  Requires Pyro Name Server to be running.
gamma.py -- Perform gamma test
quit_server.py -- Use this to quit cal_server
cal_helper.py -- Used by gamma test

In addition to being a generally useful utility, this is the best
Vision Egg demo application for demonstrating how the Vision Egg is
meant to be used in an experimental context.
